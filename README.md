# GeoGuide-AI

An AI That can Help you in Navigations using your real-time location

# Dependencies

## Node.js

Copy this repo
```
git clone https://github.com/Mxfil-G/GeoGuide-AI
```

Go into your cloned directory

```
cd GeoGuide-AI
```

Install Libraries

```
npm install

```

Run the app (After Libraries Installation is done)
```
npm start
```
and there you have, your own GeoGuide AI
